# Selenium-QAPractice-Ecomm-E2E-Framework

### This is a full-fledged framework with following key features:
1. Page Object Model within Selenium Java-TestNG Framework

2. Has 4 layers in the framework =>
  >- BDD Steps in Feature File
  >- Pagewise Step Definitions
>- Pagewise Helper Methods
>- Page Objects containing ```By Locators```
3. Wait Utils containing wrapper methods for clickability/ visibility/ invisibility checks

4. ***config.properties*** file to contain ```browser``` and ```baseURL``` details
- BaseURL: https://practice.qabrains.com/ecommerce/login
